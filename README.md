#  * Метрические алгоритмы классификации

Имеется пространство объектов X и конечное множество имён классов Y .
На множестве X задана функция расстояния ρ : X×X → [0,∞).
Существует целевая зависимость y∗ : X → Y , значения которой известны только на объектах обучающей выборки![1](https://user-images.githubusercontent.com/43415122/46093396-e33adc80-c1bf-11e8-8f8d-179c7576fc9a.png)
 Требуется построить алгоритм классификации
a : X → Y , аппроксимирующий целевую зависимость y∗(x) на всём множестве X.
